Real-Time Chat Application (MERN Stack + Socket.io)

Live demo https://github.com/uday-bhatiya/chat-app
Github repo https://github.com/uday-bhatiya/chat-app

A secure and real-time messaging platform that enables seamless communication between users. It supports one-on-one private chats as well as group conversations, allowing users to share messages, images, and files in a fast and intuitive interface.
The platform includes:

  • Real-Time Messaging using WebSockets (Socket.io).
  • User Authentication with JWT and protected routes.
  • Media Sharing (images, videos, and files).
  • User Presence Indicators (online/offline status).

Tech Stack: MongoDB, Express.js, React, Node.js, Socket.io, Redux Toolkit, Tailwind CSS
